# MyPortfolio

Abdulsamed Topcu personal portfolio frontend built with Vite and React.

## Structure

- `/src`: React application source
- `/public`: static assets
- `VITE_API_URL`: backend API base URL

## Local development

```powershell
npm install
npm run dev
```

## Production checks

```powershell
npm run lint
npm run build
```

## Notes

- This repository contains the frontend application only.
- Backend is managed separately.
